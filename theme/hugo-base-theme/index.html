Hugo base theme

Hugo theme based on gohugo

How to use

  • Place a image in static/img/
  • Create posts to display your services. Use the follow as an example:

post

+++
date = 2014-10-13T20:07:19Z
draft = true
title = "First"
description = "Ceci est mon premier post \n bla bla bla"
weight = 1
icon = "pacman"
+++

Hello first post

carrousel

+++
date = 2014-10-13T19:54:21Z
draft = true
title = "About"
weight = 5
icon = "circlestar"
[[carousel]]
  quote = "First quote"
  author  ="John Do"
  url = ""
  quote_date = "13 August"
[[carousel]]
  quote = "Second quote"
  author  ="Nhoj Od"
  url = ""
  quote_date = "14 August"
+++
